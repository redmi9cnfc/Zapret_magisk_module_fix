Используйте **KsuWebUI**.

1. При запуске разрешите **root**.
2. Выберите **zapret**.

### ℹ️ Вкладки

**🏠 Главная**
- запуск / остановка **zapret**
- включение / отключение **автозапуска при загрузке Android**

**🌐 Домены**
- добавление и удаление доменов  
- редактирование:
  - `autohosts.txt`
  - `ignore.txt`

**⚙️ Конфиг**
- редактирование `config.txt`

**🔎 Проверка**
- скрипт проверяет соединение со всеми доменами из  
`autohosts.txt` используя текущую стратегию.

# 🖥 Windows

### Основные файлы

**`start_now.cmd`**  
Запускает zapret в окне.  
Серверы ускоряются, пока открыта **командная строка**.

**`start_stop_service.cmd`**  
Создает или удаляет **службу Windows**.

При создании службы:
- zapret запускается **без окна**
- автоматически запускается **при старте Windows**

---

### Файлы конфигурации

**`autohosts.txt`**  
Список доменов сайтов с устаревшими серверами.  

⚠️ В конце файла **обязательно должна быть пустая строка**.

**`ignore.txt`**  
Список доменов **неограниченных сайтов**.

Если сайт не работает — добавьте его домен сюда.

**`check.cmd`**  
Проверяет доступ к сайтам из `autohosts.txt`.

**`config.txt`**  
Конфигурация **winws**.
