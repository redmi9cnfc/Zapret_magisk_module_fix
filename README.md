# 🖥 Windows

## Основные файлы

### `start_now.cmd`

Запускает **zapret** в окне.

Сервера будут ускоряться **пока открыта командная строка**.

---

### `start_stop_service.cmd`

Создаёт или удаляет **службу Windows**.

Если служба создана:

- zapret запускается **без окна**
- автоматически стартует **при запуске Windows**

---

# 📂 Файлы конфигурации

## `autohosts.txt`

Список доменов сайтов с устаревшими серверами.

⚠️ **Важно**

В конце файла обязательно должна быть **пустая строка**.

---

## `ignore.txt`

Список **неограниченных сайтов**.

Если сайт не работает — добавьте его домен сюда.

---

## `check.cmd`

Проверяет доступ к сайтам из:

```
autohosts.txt
```

---

## `config.txt`

Конфигурация **winws**

Вместо путей используются переменные:

```
{hosts}
{ignore}
{ipset}
{youtube}
{quicgoogle}
{tlsgoogle}
```

---

# 🐧 Linux

## Требования

Убедитесь что установлены пакеты:

```
iptables
ipset
```

Если они не установлены — установите их через пакетный менеджер.

---

## Установка

1. Откройте терминал
2. Перейдите в папку где распакован архив
3. Запустите:

```bash
./install.sh
```
