Android (Magisk)
Используйте веб-интерфейс KsuWebUI.
При запуске разрешите ему root, выберите zapret.
ℹ️На вкладке Главная можно останавливать/запускать zapret, включать/отключать автозапуск при загрузке Android.
ℹ️На вкладке Домены можно добавлять/удалять домены, редактируя autohosts и ignore
ℹ️На вкладке Конфиг можно отредактировать конфиг
ℹ️На вкладке Проверка скрипт проверит соединение ко всем доменам из autohosts.txt с текущей стратегией.

Windows
ℹ️start_now.cmd — запустить в окне, сервера будут ускоряться пока открыта командная строка
ℹ️start_stop_service.cmd — создать или удалить службу. При создании службы, zapret будет запущен без окна и будет автоматически запускаться при запуске Windows.
ℹ️autohosts.txt — список доменов сайтов с устаревшими серверами. Обязательно должна быть пустая строка в конце файла.
ℹ️check.cmd — проверить доступ к сайтам из autohosts.txt.
ℹ️ignore.txt — список доменов неограниченных сайтов. Если не работает неограниченный сайт, добавьте его домен сюда.
ℹ️config.txt — конфиг winws. Вместо путей к файлам используйте {переменные}.

ℹ️Откройте терминал в папке, куда архив был распакован
ℹ️Откройте ./install.sh
